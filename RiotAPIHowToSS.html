So What's Next?

The examples used in this guide are quite simple and not as aesthetically pleasing as one may wish. That does not mean the option is not there. In fact Riot Games gives developers access to certain graphics that can be used along with the APIs. For example at the bottom of the Ranked Info page there is a downloadable zip of all the tier icons for you to use. The idea is that with enough experience you can use Riot Games API to create websites similar to OP.GG (Tip* Type in my summoner name), or more famously LOLKING.NET.

Hope you found this guide useful and enjoy your time coding!
