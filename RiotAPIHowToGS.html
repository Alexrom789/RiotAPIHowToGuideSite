Getting Started

First thing is first, you will need to have a League of Legends account in order to register for an API key. This is to make sure no one exploits the system. If you are reading this guide, then you probably already have a LoL account, but if not you can sign up for one here at the League of Legends webpage.

I have an account, now what?

Once you have an account navigate back to the Riot Games Developer page and sign in. The sign in page will look like the following:

After logging in you will be taken to your development API key usage page which has your personal API key listed. It is important to note that this key should be kept private and not shared. If your key somehow falls in the wrong hands it can be exploited and you could possibly have you League of Legend account suspended. Below is a screenshot of where you can find your personal key. Mine has been blacked-out for security reasons.

With your API key aquired, you can now begin to use Riot Games' APIs. You can find a full list of all availible API references and their descriptions on the API methods webpage. For now though, let's head on over to the next page and we will go over a simple example on how to use the summoner lookup API.

**NOTE** In order to properly use the APIs by following the examples, you may need to install a special extension for your browser or you will get a "No 'Access-Control-Allow-Origin' header" error. I used the Chrome "Allow-Control-Allow-Orgin*" extension.
